# agrippa
A task management and documentaton system.

## Requirements

* Maven 3.5.+
* Java 11+


## Build

* Open a shell.
* Change to project directory.
* Type ``mvn clean install -Pproduction`` and press enter.


## Run

* Open a shell.
* Change to project directory.
* Type ``java -jar gui-web/target/agrippa-gui-1.1.0.jar`` and press enter.
